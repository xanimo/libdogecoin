This function derives the BIP 44 extended private key from a master private key.

BIP 44 is a standard for deterministic wallet generation, which allows for the generation of a hierarchy of keys starting from a single seed. The extended private key is a specific key in this hierarchy that is derived using BIP 44 constants and the input parameters.

Parameters
master_keyThe master private key to derive from.
accountThe account number to use in the BIP 44 key derivation.
address_indexThe address index to use in the BIP 44 key derivation.
change_levelThe change level to use in the BIP 44 key derivation, either "external" or "internal".
custom_pathA custom BIP 44 keypath to use for the derivation (optional).
keypathA string buffer to store the BIP 44 keypath used for the derivation.
bip44_keyThe BIP 44 extended private key to be generated.
Returns
0 if the key is derived successfully, -1 otherwise.
